# BiblioDrop-Backend

## Description:
It is an amazing website where people can borrow and read books. This is the backend of the website where I used CRUD operations to maintain the website. There are various kind of operations such as POST books, DELETE books, PATCH books to edit the informations, and GET the books to show the users.

## Functionalities:
Here CRUD operations are used for:
- Posting, editing, deleting, and getting books.
- Posting, editing, deleting, and getting reviews.
- Editing, deleting, and getting Users information (Editing roles etc.)
- A simple use of aggregation pipeline to check if the reviewer is a person who purchased the book or not.
- Pagination,Filtering,Sorting from the backend etc.
## Technologies:
- Node.js
- Express.js
- MONGODB
- CORS
- DOTENV
- JWT authorization
